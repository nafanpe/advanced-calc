# Advanced Calculator

A simple command-line advanced calculator built with Node.js.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Square
- Factorial
- Power calculation
- Simple interest
- Compound interest
- Square root
- Nth root

## Installation

1. Install Node.js if it is not already installed.
2. Clone or download this repository.
3. Install dependencies (none required beyond Node.js, but run this if needed):

```bash
npm install
```

## Usage

Run the calculator with Node:

```bash
node calc.js
```

Follow the prompts to select an operation and enter the required values.

## Supported Operations

When prompted, enter a number for the desired operation:

1. Addition
2. Subtraction
3. Multiplication
4. Division
5. Square
6. Factorial
7. Power
8. Simple Interest
9. Compound Interest
10. Square Root
11. Nth Root

## Notes

- Division by zero is handled with an error message.
- The calculator uses iterative methods for root calculations.
- Enter values as numbers when prompted.

## Example

```bash
node calc.js
```

Then choose an operation and input values when asked.
